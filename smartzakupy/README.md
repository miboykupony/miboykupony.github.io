# Miboy SmartZakupy – wersja premium startowa

## Gdzie wkleić

Skopiuj cały folder `smartzakupy` do:

`AdmitadBot\site\smartzakupy`

Po skopiowaniu powinno być:

- `AdmitadBot\site\smartzakupy\index.html`
- `AdmitadBot\site\smartzakupy\assets\style.css`
- `AdmitadBot\site\smartzakupy\assets\products.js`
- `AdmitadBot\site\smartzakupy\assets\app.js`

## Co działa

- nowoczesna strona startowa,
- wyszukiwanie produktów,
- rozpoznawanie budżetu z zapytania,
- sortowanie wyników według trafności,
- szybkie podpowiedzi,
- kategorie,
- responsywny wygląd,
- przykładowe karty produktów.

## Co podmienić później

W pliku:

`assets\products.js`

podmień:

- nazwy,
- ceny,
- opisy,
- tagi,
- `url: "#"` na prawdziwe linki afiliacyjne.

## Ważne

To nadal wersja bez płatnego API AI. Wygląda jak gotowa aplikacja, ale wyniki są liczone lokalnie na podstawie bazy produktów.
